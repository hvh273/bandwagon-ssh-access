## 搬瓦工的"登录"其实分三层

先把概念说清楚。搬瓦工网线怎么登录，准确来说涉及三个不同的入口：

1. **搬瓦工账号后台**（Client Area）：管理订单、续费、查看已购 VPS 的地方
2. **KiwiVM 控制面板**：搬瓦工自研的 VPS 管理界面，可以开关机、重装系统、查看 IP 和端口、重置 root 密码
3. **SSH 登录**：用终端工具直接连接到 VPS，才是真正"进入服务器"的那一步

三者是顺序关系。先进账号后台 → 找到 KiwiVM 入口 → 在 KiwiVM 里取得登录凭证 → 用 SSH 客户端连接。跳步骤的话大概率会卡住。

## 第二步：进入 KiwiVM 控制面板

在 My Services 页面，每台 VPS 右边有两个按钮——

- **Manage**：看 VPS 基本信息、升降配
- **KiwiVM Control Panel**：进管理面板

点 **KiwiVM Control Panel** 就进去了。

第一次进去可能会问你设置一个独立的 KiwiVM 密码。这个密码和账号密码是分开的，和之后 SSH 用的 root 密码也不是同一个。三个密码，三套体系，不要搞混。

进入面板之后，左侧导航就是所有可操作的功能，最常用的就这几个：

- **Main controls**：开机 / 关机 / 重启
- **Root password modification**：重置 SSH 的 root 密码
- **Install new OS**：重装系统
- **Migration**：迁移机房

## 第三步：获取 SSH 登录所需的三个信息

SSH 登录需要三样东西：**IP 地址 + 端口号 + root 密码**。缺一不可。

### 1. IP 地址和端口号

在 KiwiVM 面板首页（Main controls 页面），直接就能看到：

- **IP address**：你的服务器 IP
- **SSH Port**：注意，搬瓦工的端口**不是 22**，是随机分配的，通常是五位数，比如 28388。每次重装系统这个端口会重新随机生成。

把 IP 和端口号先记下来或者复制到别处。

### 2. 获取 root 密码

这里有个小坑要说清楚：**搬瓦工不会主动把 root 密码发到你邮箱**。

密码在重装系统的时候会出现一次，就那一次，如果没记住就只能重置。重置的方法：

1. 在 KiwiVM 面板点 **Root password modification**
2. 点 **Generate and set new root password**
3. 页面上会出现新密码——**只出现这一次，赶紧复制**

重置密码前需要保持 VPS 是**开机状态**，这点和早期版本不一样，注意别搞反。

## 第四步：用 SSH 客户端连接

三个信息都有了，接下来根据你用的系统走对应的方法。

### Windows 用户

推荐用 **Termius**，免费版够用，界面比老工具直观很多。

填入：
- Host：VPS 的 IP 地址
- Port：KiwiVM 里显示的那个端口号（不是 22）
- Username：`root`（固定是这个，全小写）
- Password：刚才复制的 root 密码

连接成功会进入命令行界面，看到 `[root@bwg ~]#` 这样的提示符就说明进去了。

### macOS / Linux 用户

打开终端，直接一行命令搞定：

bash
ssh root@你的IP地址 -p 端口号


举例：

bash
ssh root@123.123.123.123 -p 28388


输入密码（Linux 终端输密码时屏幕上不显示任何字符，这是正常的，不是没输进去），回车就行。

## 连不上 SSH 时的排查顺序

说实话，新手第一次连接失败的概率不低。按这个顺序排查基本能定位到问题：

**排查步骤：**

1. **确认 VPS 是开机状态**——在 KiwiVM 面板 Main controls 里看一眼 Running status
2. **端口号对不对**——一定要用 KiwiVM 里显示的那个，不是 22
3. **密码有没有空格**——复制 root 密码时可能带了首尾空格，手动输一遍试试
4. **IP 是否被封**——在国内部分地区，搬瓦工 IP 可能被封。可以找在线 Ping 工具测一下 IP 是否在境外能 ping 通
5. **防火墙问题**——VPS 刚装好一般没有额外防火墙，但如果你之前操作过 iptables，检查一下规则
6. **重置密码再试**——有时候密码没记对，直接在 KiwiVM 里重置一次最省事

大部分情况下，第 2 步端口不对是最高频的原因。

## FAQ：搬瓦工登录常见问题

**Q：搬瓦工网站打不开怎么办？**
A：搬瓦工主域名在国内部分网络下有访问限制。搬瓦工官方有几个镜像域名（bwh88.net、bwh81.net 等），功能和主站完全一样，换一个试试就行。

**Q：SSH 端口是什么？我用 22 连不上。**
A：搬瓦工的 SSH 端口是随机生成的，不是默认的 22，每次重装系统还会重新随机。正确端口在 KiwiVM 面板首页的 Main controls 里查看。

**Q：root 密码在哪里找？**
A：搬瓦工不发密码邮件。进 KiwiVM 面板 → Root password modification → 点击重置，把新密码立刻复制好。这个密码只显示一次。

**Q：KiwiVM 密码和 root 密码是同一个吗？**
A：不是。搬瓦工账号密码、KiwiVM 面板密码、SSH root 密码，三个是完全独立的。第一次用建议分别记下来。

**Q：我可以在 KiwiVM 里直接操作 VPS 吗，不用装 SSH 客户端？**
A：可以，KiwiVM 内置了 Root shell basic 和 advanced 两个网页端命令行工具。但功能相对有限，跑简单命令没问题，复杂交互操作还是建议用本地 SSH 客户端。
